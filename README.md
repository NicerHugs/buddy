# Accountabilibuddy
An app in progress for users to help keep one another accountable for fitness goals

## Routes
Login
Add activity
Activities
	- weekly view of all users goals

## Data
```js
{
	session: {
		name: 'Jess'
	},
	activities: [
		{
			description: 'Dance',
			user: 'Jess',
			date: '2017-01-20T03%3A01%3A10.727Z'
			complete: false
		}
	]
}
```

## Actions
login
logout
completeActivity
addActivity
